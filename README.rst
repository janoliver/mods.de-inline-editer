====================
 pOT Inline editing
====================

Diese Chrome Erweiterung zeigt bei Doppelklick auf einen Post
in diesem Post ein Eingabefenster zum Bearbeiten eben dessen
an (sofern erlaubt). Es kann dann ohne Neuladen der Seite
der Post editiert und gespeichert werden. Besonders bei 
Rechtschreibfehlern, die niemand sehen soll, sehr gut geeignet.

Unter Downloads findet ihr die Extension. Grüße, Oli
